## 什么是信息素养  
主要由**信息意识、信息知识和信息能力**组成。
| 序号 | 信息素养组成 |
| :--: | -----------: |
|  1   |     信息意识 |
|  2   |     信息知识 |
|  3   |     信息能力 |
### 具体内容:  
- *信息意识*：人的信息敏感程度。
- *信息知识*：这是信息科学技术的理论基础，又是学习信息技术的基本要求；
- *信息能力*：这是信息时代重要的生存能力，是信息素养诸要素中的核心。
